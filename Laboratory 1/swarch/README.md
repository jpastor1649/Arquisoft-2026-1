# Requirements

## System Requirements
- **Python**: 3.11 or higher
- **MySQL**: 5.7 or higher
- **Docker**: Latest version (for containerized deployment)
- **Docker Compose**: Latest version (for orchestration)

## Python Dependencies
The project uses the following Python packages (see `requirements.txt`):
- **Flask** (2.3.3) - Web framework for HTTP request handling and routing
- **Flask-SQLAlchemy** (3.1.1) - SQLAlchemy integration with Flask for ORM functionality
- **mysqlclient** (2.1.0) - MySQL database driver for Python

## Database Requirements
- MySQL server running and accessible
- Database credentials properly configured in `config.py`
- Sufficient permissions to create and manage database tables
